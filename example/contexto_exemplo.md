# Exemplo de Contexto (RAG)

Este é um arquivo de exemplo para indexação de contexto.

## Projeto

- Nome: Projeto de decisão de crédito e veículos
- Objetivo: responder perguntas com base em políticas internas

## Regras gerais

- Sempre justificar decisões com base nas regras carregadas.
- Evitar suposições fora do contexto disponível.
- Se faltar informação, sinalizar explicitamente.

## Termos importantes

- proposta
- score
- elegibilidade
- limite
- financiamento
- veículo
